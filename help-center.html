TECHNICAL/FUNCTIONCAL ISSUES

1. ) Technical/functional issues are one of the major issues in every medium, particularly to our website. This occurs when the coding is unresponsive or not properly reviewed. If any issues or bugs occur, please directly message us via the form on Feedback. Include pictures to help us easily detect the problem, such as issues or bugs.

A. If you encounter website that say “404 error” or don't open, the functionality of the hyperlinks doesn't work: faulty coding, links don't match, or unavailability of the webpage. It often happens when the webpage is not refreshed or is still searching for updates.

B. If elements won't show, it might be caused by unmatched names of files to show, unavailability of the element links from the media, or removal from the source. It can also happen when the device can't handle such a higher resolution or beyond the limit. Try to refresh or reopen the webpage; it often happens when the webpage is not refreshed or is still updating.

C. If freezing, there are several factors why it happens. It is possible for low/weak internet speed or connection, high graphics, or outdated devices. If none of these is the reason, try to refresh your browser or reopen the webpage; it took several times for the webpage to search for updates from the source itself.

D. Outplaced buttons, hyperlinks, elements, or selections are caused by an unresponsive code structure or device's pixel ratio. There are only a few chronic solutions for this matter, but reaching us can help the website to fix the problem immediately.

E. Buttons won't respond? Maybe scripts are not responding very well. It is either caused by flawed hyperlinking or misplaced or typoed code structures. This caused the buttons not to respond very well or worse.
